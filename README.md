# forklift
Projet Jeu sérieux simulateur de chario élévateur.


## 🚀 Comment lancer le jeu ?
Il existe deux manières principales de découvrir le projet.

### 🌐 Option 1 : Jouer directement dans le navigateur
C'est la méthode la plus simple.

Rendez-vous sur la page officielle : petitcyp.itch.io/forklifter.

Cliquez sur le bouton Run Game.

Une fois le chargement terminé, choisissez votre mode :

Partie Théorie : Appuyez sur le bouton correspondant pour accéder au module ActivePresenter.

Partie Simulation : Appuyez sur le bouton correspondant pour monter dans le chariot.

### 💻 Option 2 : Lancer le jeu en local
Si vous souhaitez lancer le projet depuis vos fichiers, vous aurez besoin de Visual Studio Code.

Ouvrir le projet : Lancez VS Code et ouvrez le dossier export_web du projet.

Installer l'extension : * Cliquez sur l'icône Extensions dans le menu de gauche (le carré avec un petit bloc qui s'en détache).

Recherchez l'extension "Live Server" de Ritwick Dey.

Cliquez sur Install.

Démarrer le serveur : * En bas à droite de la fenêtre VS Code, cliquez sur le bouton Go Live (avec l'icône d'antenne).

Votre navigateur s'ouvrira automatiquement. Si ce n'est pas le cas, allez à l'adresse suivante : http://127.0.0.1:5500/index.html.

Arrêter le serveur : Cliquez sur le bouton Port: 5500 en bas à droite pour stopper la diffusion.
